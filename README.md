# Бонусный фонд · TruckStop Z-463 · Кафе

Live-дашборд для TV в офисе кафе. Подключён к Google Sheets, обновляется каждый час.

## Что внутри

- `index.html` — единственный файл, рендерит весь дашборд
- Подключается к Google Sheets через gviz JSON (без API-ключей, без Apps Script)
- Автообновление каждые 60 минут плюс при возврате на вкладку

## Конфигурация

В начале `<script>` блока:

```js
const CONFIG = {
  SHEET_ID: "1okoktKxsoPDX7K3q8nG-vPLxCKSyLmud",
  SHEETS: {
    calc:    "БФ_Калькулятор",
    summary: "Итог",
    mgmt:    "Управление_5%",
    innov:   "Инновации_15%"
  },
  REFRESH_MIN: 60
};
```

Чтобы дашборд работал в Live-режиме, таблица должна быть открыта на чтение: Поделиться → «Все у кого есть ссылка» → Читатель.

## Запуск на TV

Открыть URL в kiosk-режиме Chrome:

```
chrome.exe --kiosk --noerrdialogs --disable-infobars URL
```

Без подключения к Sheets дашборд работает в Demo-режиме на встроенных данных за апрель 2026.
